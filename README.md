# TBEarly
TBEarly is a digital health platform developed to accelerate early Tuberculosis (TB) detection by addressing the challenges of delayed diagnosis and social stigma in Indonesia. A multi-model pipeline was engineered, integrating a Gradient Boosting Regressor for clinical risk classification and an NLP-based Healthcare Chatbot for adaptive user interaction. The platform streamlines the user journey from initial symptom screening to medical referral through a logic-driven interface built as a functional Minimum Viable Product (MVP). By translating complex epidemiological needs into a scalable digital solution, this project demonstrates a commitment to enhancing public health accessibility and supporting SDG 3 (Good Health and Well-being).
